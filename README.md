**SpendSantai** is a lightweight, intuitive personal finance tracker designed to help you manage your money with ease. True to its name (*Santai* meaning "relaxed" in Malay), this app removes the complexity from budgeting and focuses on what matters: tracking where your money goes.

---

## ‚ú® Features

* **Effortless Tracking:** Log your income and expenses with just a few taps.
* **Smart Categorization:** Organize spending into custom categories like Food, Transport, and Utilities.
* **Visual Insights:** Understand your habits through clean, easy-to-digest charts and summaries.
* **Budget Management:** Set monthly limits to ensure you never overspend.
* **Privacy First:** Data is stored locally on your device, ensuring your financial information stays yours.
* **Dark Mode Support:** Easy on the eyes for those late-night budget checks.

## üöÄ Tech Stack

* **Framework:** Flutter / Dart
* **Database:** SQLite (via sqflite) or Hive for local persistence
* **State Management:** Provider / Riverpod
* **UI Components:** Material Design 3

## üõ†Ô∏è Getting Started

### Prerequisites

* Flutter SDK installed on your machine.
* An Android/iOS emulator or a physical device.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/hzqfarhan/SpendSantai.git](https://github.com/hzqfarhan/SpendSantai.git)
    cd SpendSantai
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the application:**
    ```bash
    flutter run
    ```

## üìñ How to Use

1.  **Initialize:** Open the app and set your monthly budget or starting balance.
2.  **Add Entry:** Tap the floating action button to record a new expense.
3.  **Review:** Use the dashboard to see your remaining balance and top spending categories.
4.  **Adjust:** Modify categories or budget limits in the settings tab.
